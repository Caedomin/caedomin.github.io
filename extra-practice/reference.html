How to Resize Images 

When working with images that will be placed on a Divi page, you will want to optimize the width and file size depending on how you are going to use the image.

  • Logo or Icons: Set width to 200–400px (get under 20kb)
  • Regular Image : Set width to 800px (get under 100kb)
    • A regular image placed in the Divi Page Builder will shrink and grow depending on the size of the browser window. We set this to 800px because this is the largest these images will get.
  • Full-width Image : Set width to 1000–1500px (get under 200kb)
    • A full-width image placed in the Divi Page Builder will span the whole page, which can be as small as a mobile screen and as large as a monitor can go. We will set these images to somewhere between 1000–1500px to help make sure the images don’t get pixelated on the large screens. You still want to optimize these images so that the page can load as fast as possible. You should also limit the number of full-width images on a page for this same reason.
